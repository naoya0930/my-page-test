# spec5: UI/UX改善および機能追加仕様書 (ホーム・統計画面の改修)

## 1. `/statistics`（統計画面）の改修
* **ヒートマップへの現在日付の表示**:
    * アクティビティヒートマップ上において、現在の日付（本日）のセルを視覚的に特定できるようにする。
    * 具体的な実装例：本日のセルの枠線をアクセントカラーで強調する、特別なインジケーター（小さなドットなど）を付与する、あるいはホバー時のツールチップに「今日」と明記するなど。

## 2. `/home`（ホーム画面）のUI・テキスト改善
* **今週の実行状況における本日の強調**:
    * 「今週の実行状況」を表示するセクションにおいて、現在の日付（本日）に該当する曜日・日付の箇所をハイライトする。
    * 具体的な実装例：本日の要素のみ背景色をプライマリカラーの薄い色にする、テキストを太字にする、または「今日」を示す小さなバッジを付与する。
* **過去週のテキスト削除**:
    * 過去の週のデータを表示している際に表示されていた「(閲覧のみ)」というテキストを削除し、UIをクリーンに保つ。
* **統計画面へのリンク化**:
    * 過去の週のデータの「過去の記録を閲覧できます」というテキストをクリッカブルなテキストリンク（またはボタン）に変更する。
      * その際に押下時に `/statistics`（統計画面）へ遷移するようにルーティングを設定する。
* **「未完了」ラベルの非表示**:
    * 「モーニングページ」の「未完了」というテキストを削除にする。

## 3. `/home`（ホーム画面）の達成時セレブレーション演出
* **対象となる条件**:
    * 以下の **いずれか（OR条件）** を満たした場合に演出を発動させる。
        1. 「モーニングページ」の進捗が `7/7`（コンプリート）になった。
        2. 「アーティストデート」が完了（実施済み）になった。
* **演出の仕様（めでたい感じの表現）**:
    * 該当条件を満たした項目のカードやセクションにおいて、視覚的な達成感（めでたい感じ）を演出する。
    * 具体的な実装例：
        * 該当要素の背景色を特別感のある色（薄いゴールドやグラデーションなど）に変化させる。
        * チェックマークを豪華なアイコン（王冠、キラキラ、クラッカーなど）に変更する。
        * 達成状態になった瞬間に、CSSアニメーション（紙吹雪、要素のバウンス、パーティクルなど）をワンポイントで再生する。
